Links úteis
Aqui está uma lista de links que podem ser usados para ajudar no andamento da aula:

Todos os atalhos de teclado do VS Code: https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf
Comentar sobre os atalhos de teclado que serão usados pode ajudar os alunos a ganhar velocidade durante as aulas. Isso vai fazer com que os alunos sejam mais eficientes no fim do projeto do que no início dele.

Lista de abreviações Emmet: https://docs.emmet.io/cheat-sheet/
As abreviações Emmet são usadas para gerar códigos HTML de forma mais rápida. Elas podem ajudar muito no desenvolvimento. Ensinar os alunos a usar esse recurso pode poupar tempo, pois o código gerado automaticamente não terá erros de syntax como aspas esquecidas e tags fora de lugar.

Princípios de design de software: https://pt.linkedin.com/pulse/princ%C3%ADpios-de-design-software-explorando-solid-dry-e-alves-junior
Ao construir o projeto, utilizou-se o princípio de design SOLID. O livro comenta a respeito do DRY e sobre a procura de não se repetir durante o desenvolvimento do software.
